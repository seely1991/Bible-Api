New King James Version API

This web api is used to retrieve any verses from the New King James version of the Bible using the following format:

[url]/api/get_verse/:book/:chapter/:verse(s)

or for retrieving entire chapters:

[url]/api/get_chapter/:book/:chapter(s)

for instance:

https://functional-pancreas.glitch.me/api/get_verse/John/3/16-18
